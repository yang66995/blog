安装hexo
```ssh
$ npm install -g hexo-cli
```
创建新文件夹存储博客文件，名称随意
```ssh
mkdir <your_blog_name>  
```
在文件夹下`git bash`往文件夹里安装Hexo
```ssh
hexo init
```
